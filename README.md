# Nuxt 

## Ringkasan Instalasi
1. cd nama-project
<br>
2. npm install
<br>
3. npm run dev

## Setup
Make sure to install dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run dev
```
