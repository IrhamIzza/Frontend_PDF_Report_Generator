# Nuxt 

## Ringkasan Instalasi
1. git clone https://github.com/IrhamIzza/Frontend_PDF_Report_Generator
2. cd nama-project
3. npm install
4. npm run dev

## Setup
Make sure to install dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run dev
```
